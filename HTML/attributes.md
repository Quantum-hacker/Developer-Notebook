# Common Attributes

id, class, src, href, alt, title, target, type

# Common HTML Attributes

- `id=""` — Unique identifier
- `class=""` — CSS or JS hook
- `src=""` — Image or script file path
- `href=""` — Link target
- `alt=""` — Image description
- `title=""` — Tooltip text
- `target="_blank"` — Open in new tab
- `style=""` — Inline CSS (avoid)
- `type=""` — Input or script type
