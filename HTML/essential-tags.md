# Essential HTML Tags

## Structure Tags
- <!DOCTYPE html> — Declares HTML5
- <html> — Root of the document
- <head> — Metadata, title, links, scripts
- <body> — Visible page content

## Text Tags
- <h1>–<h6> — Headings
- <p> — Paragraph
- <span> — Inline text container
- <strong> — Bold
- <em> — Italic

## Media Tags
- <img> — Image
- <video> — Video player
- <audio> — Audio player

## Link & Navigation
- <a> — Link
- <nav> — Navigation area

## Lists
- <ul>, <ol>, <li>

## Containers
- <div>, <section>, <article>, <header>, <footer>, <main>
